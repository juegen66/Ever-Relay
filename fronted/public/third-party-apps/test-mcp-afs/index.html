Local MCP Test

Test MCP AFS

This app is a host-side test window for backend MCP loading. When this window is focused, Copilot switches to third_party_agent with app slug test_mcp_afs.

  1. Start the local MCP HTTP server from the sibling everrelay-third-party-sdk repo.
  2. Bind test_mcp_afs to http://127.0.0.1:3310/mcp.
  3. Focus this window and ask Copilot to use or list MCP tools.
Remote MCP tools will be exposed to the agent as mcp_test_mcp_afs__*.
